## GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to `Settings -> Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose `main` and `/root`.
6. Save and wait for the site URL.
